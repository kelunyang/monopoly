# JavaScript 多人連線大富翁

*** 本程式後端使用Node.js ***

功能
====

1. 即時多人連線遊戲（回合制）
2. 自訂地圖，可自行上傳Excel-XML
3. 時代升級，設定在N回合後可以切換時代，提供更多升級物品、地圖
4. 使用Node.js+jQuery，檔案約1M，效率較高

系統需求
=======

1. Server: Node.js 8 以上（必須支援ES6，mysql操作需要支援promise）
2. Client: Chrome / Firefox / Edge ，不可以使用IE
